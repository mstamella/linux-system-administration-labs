# Enterprise Linux Server Deployment

## Overview
This lab demonstrates deployment and recovery of a Red Hat Enterprise Linux 9 server.

## Environment
VirtualBox  
RHEL 9  
20 GB disk  

## Tasks Performed
- Install RHEL server
- Configure repositories
- Reset root password
- Configure journald logging
- Configure boot targets

## Skills Demonstrated
- RHEL system installation
- repository configuration
- system recovery
- systemd administration
